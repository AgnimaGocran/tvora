# Что такое «главный заваренный» и «капельник»

**Контур у основания алтаря** — отвод от магистрали конденсата печей.

1. **Магистраль конденсата**
   Толстая труба, по ней идёт вода с лёгким напором от конденсаторов.

2. **Главный запор K-17 (заварен намертво)**

   * Стальной хомут/фланец.
   * По кругу — *бусины сварки* («как приварено»).
   * Срезные болты (головы намеренно отломаны при консервации).
   * Когда-то висела свинцовая бирка/пломба — могла сгнить/исчезнуть, но сути не меняет.
   * Назначение: **полностью глушит узел**, чтобы никто не «откусывал» воду впрямую. Любая попытка вскрыть = следы реза/ударов, то есть явный саботаж.

3. **Сервисный капельник (пробоотбор/дренаж)**

   * Тонкий штуцер ≈ ¼″ с маленьким краником **за решёткой**.
   * Пломбы **нет по регламенту**: его держат для санитарной промывки/сброса воздуха/набора проб.
   * Дебит — **струйка/капли** (условно 0.1–0.3 л/мин): достаточно для корыта, но не для «водопоя».
   * Имеет собственный **слив в приямок**, чтобы первые мутные порции не лились на пол.

4. **Дренажный приямок/зольник**

   * Под очагом — зольный ящик; рядом — приямок дренажа. Удобно смывать муть и брать чистую порцию.

# Как это использует Дерния — пошагово

1. **Главный — не трогаем.** Он «намертво»: заварен и так останется. Это ваша страховка от обвинений.
2. **Открыть капельник.** Повернула микрокран на четверть оборота.
3. **Первые 200–300 мл — в сток.** Там ржавчина/слизь.
4. **Дальше — в корыто.** Набрать ровно столько, сколько нужно (ведро/полведра).
5. **Закрыть капельник.** Протереть краник золой/пылью — естественная патина.
6. **Огонь:** поджечь трут термоформой; щепа → свежая **древесная зола** для щёлока.
7. **Работа Мастера:** вода → промыв, зола+горячая вода → щёлок, соскоб, растяжка, сушка, финиш слёзным рассолом (тонко).

> Вся сцена живёт **в одном месте**: вода — из капельника, жар/зола — из старой топки. Никого звать не надо.

# Как это выглядит (чтобы ты описала без лишних слов)

* **Главный узел:** толстый хомут, по кругу — тёмные сварные «бусины», болты без голов. Если когда-то была бирка — сейчас торчит только ржавая проволока.
* **Капельник:** тонкая трубка за решёткой, крохотный вентиль; при открытии — частая дробь капель, потом тонкая струйка.
* **Звук:** «тик-тик-тик» по металлу в первые секунды, затем ровный шёпот струйки.
* **Запах:** сырой железный привкус; когда включили щёлок — мыльная горечь.

# Почему так безопасно для сюжета

* **Аудитобезопасно:** главный запор цел — докопаться «кто сорвал пломбу» не к кому. Капельник по регламенту допускается для промывок.
* **Отмазка готова:** «Санитарная промывка/пробоотбор. Расход — одно ведро. Главный узел не трогался». Проверяющему нечего предъявить без тяжёлой экспертизы (которой никто ради ведра не запустит).
* **Крюк на будущее:** если когда-нибудь главный хомут окажется сорван — это почти наверняка «работа людей/сильного демона» и повод для большой сцены.
